## 📊 דרך העבודה

### שלב 1: הכנת ה-Sheet
1. פתחי Google Sheets חדש
2. שמי לו: "Windows CLI Generator - Test Results"
3. צרי את העמודות: #, Input, Expected Output, Actual Output, תקין?, הערות

### שלב 2: הרצת הבדיקות
1. הפעילי את `app.py`
2. עברי על כל אחד מ-15 התרחישים
3. הזיני את ה-Input באפליקציה
4. העתיקי את ה-Output שהתקבל
5. הדביקי ב-Sheet ב-Actual Output

### שלב 3: ניתוח
1. השווי Actual vs Expected
2. סמני ✅ אם תקין, ❌ אם שגוי
3. רשמי הערות על הבדלים

### שלב 4: חישוב הצלחה
```
שיעור הצלחה = (מספר תקינים / 15) × 100%
```

## 🎯 קריטריונים להצלחה

### ✅ נחשב תקין אם:
- הפקודה תעבוד ב-CMD/PowerShell
- הפקודה משיגה את המטרה שנתבקשה
- הפקודה בטוחה להרצה

### ❌ נחשב שגוי אם:
- הפקודה תיתן שגיאה
- הפקודה לא משיגה את המטרה
- הפקודה עלולה להזיק (למשל: מחיקה לא בטוחה)
